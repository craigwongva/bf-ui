# Beachfront UI

## Client

### Running Locally

```
~/beachfront-client$ npm install
~/beachfront-client$ npm run watch
```

### Building

```
~/beachfront-client$ NODE_ENV=production npm run build
```

Check `dist` folder afterwards.


## Server

### Running Locally

```
$ cd server
$ go run
$
```

### Building

```
~/beachfront-server$ go get
~/beachfront-server$ go build
```
